## [Mac 1.0.0] - 2026-07-20

### Added
- Initial macOS-focused release of the Hermex fork.
- Native Mac home, multiwindow settings, keyboard commands, and Mac-specific
  session and composer behavior.
- Sessions, streaming chat, model/profile/project selection, scheduled tasks,
  skills, workspace browsing, memory, and insights against a self-hosted server.
- Universal Apple silicon and Intel Developer ID build for macOS 15 or later.
- Signed and notarized DMG packaging with GitHub Actions draft-release automation.

### Changed
- Synced with upstream Hermex through commit `77e8747`, including Kanban contract
  groundwork and chat transcript opening improvements.

### Known limitations
- Updates are manual; version 1.0.0 does not include an automatic updater.
